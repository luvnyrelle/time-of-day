Time of Day

A JavaScript Project by Nyrelle Vital

Greeting goes here!

Day of the week goes here!

This webpage changes its greeting, background color, and image based on the time of day!

Using JavaScript, it checks the current hour to figure out if it's morning (5am-12pm), afternoon (12pm-6pm), or night (6pm-5am).

Depending on the time, it shows a matching greeting, changes the background to a fitting color, and updates the image to match the time.

The page is styled with CSS to look neat and adjust nicely to different screen sizes.
